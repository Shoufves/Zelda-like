# Zelda Like — 类塞尔达动作角色扮演游戏

这是一个使用 Pygame 开发的简易 ARPG 游戏，灵感来源于《塞尔达传说》。玩家将探索地图、击败怪物、收集物品、提升属性，并不断深入更危险的楼层。

## 功能特性

- **俯视角 2D 地图**，基于网格移动和碰撞检测
- **玩家控制**：移动、攻击（左键）、魔法（右键）
- **多种敌人**：竹妖、幽灵、浣熊、鱿鱼等，每种拥有独特行为
- **楼层系统**：每层敌人更强，奖励更多经验，清空所有浣熊后进入下一层
- **背包系统**：拾取怪物掉落物，使用或丢弃道具（按 `B` 键打开）
- **经验与升级**：获得经验后可在菜单（按 `M` 键）中强化生命、能量、攻击、魔法、速度
- **魔法系统**：治疗术与火焰法术，消耗能量施放
- **A* 寻路**：敌人利用 A* 算法智能追踪玩家
- **丰富的粒子效果**：攻击、死亡、拾取等均有视觉效果
- **开始与结束界面**：支持重新开始与退出

## 如何运行

### 环境要求
- Python 3.6 或更高版本
- Pygame 2.0 或更高版本

### 安装步骤
1. 克隆本仓库到本地
   ```bash
   git clone https://github.com/yourusername/your-repo-name.git
   cd your-repo-name
   ```
2. 安装依赖
   ```bash
   pip install -r requirements.txt
   ```
3. 进入 `game_test/code` 目录并运行主程序
   ```bash
   cd game_test/code
   python main.py
   ```

> **⚠️ 注意**：请确保在 `code` 目录下运行 `main.py`，因为资源文件（音频、图片、地图数据）均通过相对路径 `'../xxx'` 引用。如果在其他位置运行，将无法加载资源。

## 游戏玩法详解

### 战斗与生存
- 屏幕左上角显示**生命值（红色）**和**能量值（蓝色）**，能量用于施法。
- 击败敌人可获得经验值，经验值显示在右下角。
- 怪物会掉落各种道具（浣熊草药、能量根、幸运符等），靠近自动拾取存入背包。
- 在背包中左键使用消耗品（如治疗），右键丢弃物品。

### 楼层机制
- 初始楼层为 1 层，地图上会生成若干浣熊（raccoon）。**必须击败所有浣熊**才能进入下一层。
- 每深入一层，怪物属性提升，经验奖励增加。
- 进入新楼层时，玩家会恢复少量生命。

### 升级系统
- 按 `M` 键打开升级菜单，使用经验值提升五项属性：
  - **Health**：最大生命值
  - **Energy**：最大能量值
  - **Attack**：基础攻击力（叠加武器伤害）
  - **Magic**：魔法强度（叠加法术伤害）
  - **Speed**：移动速度
- 每次升级后，下次升级所需经验会增加。

### 物品与掉落
- 每种怪物有自己的掉落表（在 `data/items.json` 中定义）。
- 浣熊怪物会掉落特殊道具，如**古代护符**可增加一次复活机会（角色死亡后自动复活并恢复 50% 生命）。
